# Projeto de Containerização com Docker - EBAC

Este repositório foca no uso do Docker para isolamento de aplicações e automação de ambientes de desenvolvimento através de infraestrutura como código.

## O que foi realizado:
* **Criação de Dockerfiles:** Configuração completa de imagens para padronização de ambientes.
* **Gestão de Contêineres:** Criação, execução e monitoramento de instâncias isoladas.
* **Eficiência de Build:** Estruturação de camadas de imagem para otimizar o armazenamento e a velocidade de implantação.
* **Padronização de Dependências:** Garantia de que todas as bibliotecas e o runtime estejam presentes dentro do contêiner.

## Tecnologias:
* Docker
* Dockerfile
* Linux / Bash
* Git / GitHub
